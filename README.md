# User Dashboard

A modern user dashboard built with **React** and **Tailwind CSS**, with the primary goal of learning and implementing **authentication from scratch** using **JWT** and **HTTP Cookies**.

> **Note:** This project is currently focused on authentication architecture and UI development rather than complete dashboard functionality.

---

## Features

- User Authentication (JWT + Cookies)
- Login & Signup Flow
- Protected Routes
- Modern Dashboard UI
- Responsive Design
- Profile Page UI
- Clean Component-Based Architecture
- React Router Navigation

---

## Tech Stack

- React
- React Router
- Tailwind CSS
- JWT (JSON Web Tokens)
- HTTP Cookies
- Vite

---

## Project Focus

The main objective of this project was to understand how authentication works behind the scenes without relying on third-party authentication services.

Topics explored include:

- JWT-based Authentication
- Cookie-based Session Handling
- Route Protection
- Authentication Flow
- Frontend State Management for Auth
- Building a scalable project structure

---

## Current Status

### Implemented

- Authentication flow using JWT

### Not Implemented Yet
